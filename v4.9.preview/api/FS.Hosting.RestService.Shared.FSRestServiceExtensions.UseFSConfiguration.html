UseFSConfiguration(IWebHostBuilder, string[], bool)

Fügt eine Konfiguration aus der angegebenen JSON-Datei zur Konfiguration des WebHostBuilders hinzu. Ist der Wert nicht gesetzt, wird der builder unverändert zurückgegeben.

public static IWebHostBuilder UseFSConfiguration(this IWebHostBuilder builder, string[] args, bool optional = false)

Parameters

builder IWebHostBuilder

Der zu konfigurierende Microsoft.AspNetCore.Hosting.IWebHostBuilder.

args string[]

Die Kommandozeilenargumente. Der Pfad zur JSON-Konfigurationsdatei wird aus args[1] ausgelesen.

optional bool

Gibt an, ob die Konfigurationsdatei optional ist. Bei true (Standard) wird keine Exception geworfen, wenn die Datei nicht existiert. Bei false wird eine FileNotFoundException geworfen, wenn die Datei nicht gefunden wird.

Returns

IWebHostBuilder

Der konfigurierte Microsoft.AspNetCore.Hosting.IWebHostBuilder.
