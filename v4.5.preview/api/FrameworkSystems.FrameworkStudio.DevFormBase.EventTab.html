FrameworkEventArgs

Alle FormMethod-EventHandler haben als (einziger) Parameter ein Objekt vom Typ FrameworkEventArgs (oder einer Ableitung). FrameworkEventArgs stellt selbst noch keine besondere Funktionalität bereit, dies kann aber von Ableitungen gemacht werden. So ist es möglich, dem EventHandler weitere Informationen zu übergeben. Siehe hierzu FrameworkSystems.FrameworkStudio.DevFormBase.EventTab.DevEventHandlerOnRowDoubleClickDefinition.
