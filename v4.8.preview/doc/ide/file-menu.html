File Menü

In den folgenden Abschnitten werden die Menüpunkte im Menü File beschrieben.

Exit

Framework Studio wird geschlossen - nicht gespeicherte Änderungen können davor nochmal gesichert werden.

Restart with Maintenance Mode

Falls zu Beginn Framework Studio über das Setzen der Checkbox Maintenance Mode im Login-Dialog nicht wie gewünscht im Wartungsmodus gestartet wurde, kann die Anwendung schnell und einfach hierüber erneut im Maintenance Mode gestartet werden. Sämtliche zuvor getätigte Login-Einstellungen werden dabei automatisch übernommen.

New

Anlegen einer neuen Datei in dem sich befindendem Verzeichnis.
